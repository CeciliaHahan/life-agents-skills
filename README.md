# life-agents-skills

Personal agent skills for daily life automation.

## Skills

| Skill | Description |
|-------|-------------|
| [bookkeeping](skills/bookkeeping/SKILL.md) | 每周日记账流程：读取交易截图，提取数据，更新到 Excel |
| [emerge](skills/emerge/SKILL.md) | 识别正在聚合成更大东西的 pattern clusters，找到可以变成项目/文章/产品的想法群 |
| [ideas](skills/ideas/SKILL.md) | 扫描 Obsidian vault 中的笔记，发现 emerging patterns，生成 idea report |
| [trace](skills/trace/SKILL.md) | 追溯某个想法在 Obsidian 笔记中的演变过程，跨 vault 搜索并生成时间线与总结 |

## Install

```bash
npx skills add CeciliaHahan/life-agents-skills
```
